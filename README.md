		執行 start.bat 
		將預設參數並畫出最短路線圖及收斂曲線圖

			runs       : 30
			iterations : 1000
			nodes      : 51
			ants       : 51
			pheromone  : 1
			alpha      : 1
			beta       : 2.8
			rho        : 0.9
			file input : eil51.txt

		[main.exe] [runs] [iterations] [nodes] [ants] [pheromone] [alpha] [beta] [rho] [file_input]
